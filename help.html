Flare / XRP Setup

This utility will setup your XRP and ETH cryptocurrency accounts to receive the Flare token (Spark) when it is airdropped. Specifically:

  • You need to specify the secret key for the XRP account you wish to setup. ONLY DO THIS ON A SECURE SYSTEM. You will receive an amount of Spark corresponding to your XRP account balance.
  • You need to specify a address to an ETH wallet to receive the Spark token.
  • After a valid XRP key and ETH address are provided, this tool will craft and submit a XRP transaction to setup your account.
  • Optionally this tool will only setup the transaction, if you wish to run this in offline mode, click the corresponding option in the settings section. YOU WILL NEED TO SUBMIT THE PRODUCED TRANSACTION ON ANOTHER SYSTEM TO COMPLETE THE PROCESS
  • If you wish to run this tool on the testnet, click the option in the settings section.

This tool will create a transaction that sets the message key on your XRP account to the provided ETH address. A snapshot of the XRP ledger will be taken for the Flare Spark airdrop after which the accounts which have been setup this way will receive corresponding tokens
